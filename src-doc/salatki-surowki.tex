\recipe{Surówka z~moreli i~bananów}

\begin{Ingred}
    \item \qty{30}{\deka\gram} moreli
    \item \num{1} jabłko
    \item \num{.5} banana
    \item \num{1} łyżka soku z~cytryny
    \item \num{1} garść rodzynek
    \item \num{1} łyżka miodu
    \item \num{.5} filiżanki śmietany \qty{12}{\percent}
    \item \num{2} łyżki posiekanych orzechów włoskich
\end{Ingred}

Owoce i~rodzynki umyć, osuszyć, jabłko i~banana obrać ze skórki, pokroić
w~drobną kostkę, skropić sokiem z~cytryny. Morele pokroić na połowy, wyjąć
pestki. W połówkach moreli układać pokrojone owoce, rodzynki i~orzechy. Polać
śmietaną, wymieszaną z~miodem

\recipe{Krabowa}

\begin{Ingred}
    \item paluszki krabowe
    \item majonez
    \item \num{1} puszka kukurydzy
    \item \num{5} plastrów ananasa
    \item ser żółty
    \item natka pietruszki
\end{Ingred}

Paluszki krabowe pokroić w~kostkę, następnie wymieszać z~kukurydzą pokrojonym
ananasem i~majonezem. Na wierzch dodać żółty ser i~natkę pietruszki.

\recipe{Sałatka z~tuńczyka}

\begin{Ingred}
    \item \num{5} jajek
    \item tuńczyk w~sosie własnym
    \item \qty{30}{\deka\gram} żółtego sera
    \item majonez
    \item szczypiorek
\end{Ingred}

Jajka ugotować na twardo, białko utrzeć na tarce na drobnych oczkach, następnie
ułożyć tuńczyka, ser majonez. Na wierzch ułożyć starte żółtko i~szczypiorek.

\recipe{Paluszki ananasowe}

\begin{Ingred}
    \item \num{1} ananas
    \item \num{4} łyżki brązowego cukru
    \item \num{.5} szklanki rumu
    \item \num{2} jajka
    \item \num{2} łyżki mąki
\end{Ingred}

Wyciąć środek ananasa, pokroić na \num{4} części. Jajka i~mąkę zmiksować, dodać
rum i~rozpuszczony cukier, polać tym paluszki ananasowe, zapiec.

\recipe{Sałatka grecka}

\begin{Ingred}
    \item \qty{100}{\gram} sera pleśniowego
    \item \num{10} pomidorów koktajlowych lub \num{2} pomidory
    \item \num{1} świeży ogórek
    \item \numrange{10}{15} oliwek
    \item \num{1} czerwona cebula
    \item \num{1} łyżka oliwy
    \item sól
    \item pieprz
\end{Ingred}

Ser pleśniowy pokroić w~kostkę. Pomidorki umyć i~pokroić w~ćwiartki lub
połówki. Ogórek obrać ze skórki i~pokroić w~kostkę. Cebulę pokroić w~piórka,
jeżeli ma być łagodna w~smaku sparzyć na sitku wrzątkiem. Oliwki odsączyć
z~zalewy, dodać oliwę.

Wszystkie składniki wymieszać. Doprawić solą i~pieprzem. Podawać z~białym
pszennym pieczywem.
